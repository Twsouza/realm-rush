# Realm Rush

Realm Rush is a tower defense game built on Unity.

## Game design

![Realm Rush Game Design](./game-overview.jpg)

It's a Sci Fi Tower Defense, the player will place the towers to defend and prevent damage to the base.

* Enemy pathing: Enemies spawn and follow best path to reach player base.
* Place twoers: Player can place towes that shoot at enemies within range.
* Enemy health: Multiple hits to be killed.
